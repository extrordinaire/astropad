## Ejercicio 6

sup A[8] = `1000_0000`$_2$ ^ B[8] = `1101_0000`$_2$ ^ C[8]:

<ol className="custom-alpha">
  <li>
    $A + B = C$
    
    $$
        \begin{array}{r}
          \bin{   1000 0000} = A \\
        + \bin{   1101 0000} = B \\
          \hline
          \bin{\ovf{1} 0101 0000} = C
        \end{array}
    $$  
    
    C = `1` `0101_0000`$_2$ $\implies$ Overflow = `1`$_2$ 
  </li>

  <li>
    $A - B = C \implies A + \cii{B} = \cii{C}$   
    $$ 
    \begin{align*}
      \overline{B} &= &\bin{0010 1111} \\
                   &+ &\bin{1} \\
                   \hline
      \cii{B}     &= &\bin{0011 0000}
    \end{align*}
    $$

    $$
        \begin{align*}
          \bin{  1000 0000} &= A \\
        + \bin{  0011 0000} &= \cii{B} \\
          \hline
          \bin{  1011 0000} &= \cii{C}
        \end{align*}
    $$

    $$
      \begin {align*}
        \cii { C}            &= \ci{C} &+ \bin { 1} \\
        \bin { 1011 0000} &= \ci{C} &+ \bin { 1} \\
        -\bin { 1}        &= &- \bin { 1} \\
        \hline
        \bin { 1010 1111} &= \ci{C} \\
        \bin { 0101 0000} &= C

      \end{align*}
    $$

  </li>

  <li>
    Obtenemos mayor rango de representacion.  
    Operamos con un solo cero.
  </li>
